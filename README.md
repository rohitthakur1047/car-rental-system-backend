# 🚗 Car Rental System (Backend)

## 🔧 Tech Stack
- Java
- Spring Boot
- MySQL
- Spring Data JPA
- Spring Security

## 🚀 Features
- Add Car
- View Cars
- Book Car
- Prevent Double Booking
- Calculate Price
